Connect from Microstrategy Desktop
==================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can use `MongoDB Connector for BI <https://www.mongodb.com/download-center?jmp=tbl#bi-connector>`_
with `Microstrategy Desktop
<https://www.microstrategy.com/us/get-started>`_.

Prerequisites
-------------

- Install the MongoDB :doc:`C Authentication Plugin
  </tutorial/install-auth-plugin-c>` for your platform.

- Create a 64-bit :doc:`system Data Source Name (DSN)
  </tutorial/create-system-dsn>`.
